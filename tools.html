Et si je veux utiliser Role + hors ligne ?

C'est fort simple ! Vous avez juste à télécharger Role + juste à le télécharger ici

Si vous êtes sur ordinateur : Vous avez juste a extraire le fichier zip dans un dossier et a lancer le fichier "index.html"

Si vous êtes sur téléphone : Vous devez extraire le fichier zip dans un dossier aussi, vous ouvrez le fichier "index.html" avec le navigateur de votre choix et vous creez un raccourci sur votre bureau. (Je précise que j'ai testé sur Android, que je n'ai pas d'iPhone chez moi pour tester, mais la technique devrait être la même normalement)
